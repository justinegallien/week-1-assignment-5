Basic Git Commands

Setting Up:
git init: Initializes a new Git repository in a directory. This is the first command you'll use to start version control for a project. git config: Sets configuration options for Git, such as your username and email. This is important for identifying your commits.
Working with Changes:
git add: Stages changes for commit. You can add specific files or all changes in a directory. git commit: Saves the staged changes with a descriptive message. This creates a new version in your project's history. git status: Shows the current state of your repository, including which files have been modified or staged. git diff: Displays the differences between your working directory and the staged changes, or between different commits.
Branching and Merging:
git branch: Lists, creates, or deletes branches. Branches allow you to work on different features or bug fixes in isolation. git checkout: Switches to a different branch. git merge: Combines changes from one branch into another.
Remote Repositories:
git clone: Creates a copy of a remote repository on your local machine. git push: Uploads local commits to a remote repository. git pull: Downloads changes from a remote repository and merges them into your current branch.
Viewing History:
git log: Shows the commit history of your project. You can customize the output to see different levels of detail.

Basic Bash Commands

Navigation and File Management
ls: Lists the files and directories in the current directory. You can use options like -l (long listing), -a (show hidden files), and -h (human-readable file sizes) to customize the output.
cd: Changes the current directory. You can specify the path to the directory you want to move to.
pwd: Prints the current working directory.
mkdir: Creates a new directory.
rmdir: Removes an empty directory.
rm: Removes files or directories. Use with caution, as this command is irreversible! Options include -r (recursive for directories) and -f (force). cp: Copies files or directories.
mv: Moves or renames files or directories.
touch: Creates an empty file or updates the timestamp of an existing file.
Viewing and Editing Files
cat: Displays the contents of a file. less: Views the contents of a file one page at a time, allowing you to navigate through the file.
head: Displays the first few lines of a file.
tail: Displays the last few lines of a file.
echo: Prints text to the terminal.
vi or nano: Text editors for creating and editing files.
Searching and Filtering
grep: Searches for patterns in files.
find: Searches for files based on various criteria.
System Information df: Displays disk space usage.
du: Displays file and directory space usage.
ps: Lists running processes.
top: Displays real-time system information, including CPU and memory usage.
uname: Displays system information.
Other Useful Commands history: Displays a list of previously used commands.
clear: Clears the terminal screen.
man: Displays the manual page for a command.
sudo: Executes a command with superuser privileges
